# Paleta de Cores

Bem-vindo ao Guia de Cores do SAPRON UI. Aqui, você encontrará a coleção de cores selecionadas para
os sistemas do sapron.

## Convenção de Nomenclatura das Cores
No SAPRON UI, as cores são nomeadas usando a seguinte convenção:

$ seguido do nome da cor em inglês
Um número de 50 a 900 para indicar variações de contraste
Essas são as cores utilizadas no SAPRON
